The motivation behind this app...

I started working on web apps since I started teaching at a polytechnic in Singapore. The apps were originally meant to be simply iPad's iBook's widgets, and subsequently were deployed as individual apps in the school's proprietary CMS.

I felt that both these two avenues were limited in their reach, and decided to deploy the apps to the Internet instead, so that more people can use them.

Looking forward to adding more apps!


This app is developed using the open-source technologies listed below:

  • AngularJS
  • Angular-UI-Router
  • Jquery
  • NodeJS
  • Express
  • RequireJS
  • Twitter Bootstrap 3.0
  • Flot
  • Fastclick

This app is currently hosted using Heroku's free plan
